## 功能特性

- ✅ 支持5种收藏类型：想看、在看、已看、搁置、抛弃
- ✅ 支持11种番剧属性的自定义选择
- ✅ 支持Excel和TXT两种输出格式
- ✅ 自动处理分页数据，无需手动操作
- ✅ 友好的命令行交互，操作简单直观

## 支持的输出属性

| 序号 | 属性名称 | 描述 |
|------|----------|------|
| 1    | 番剧名   | 番剧的原名（通常为日文） |
| 2    | 中文名   | 番剧的中文译名 |
| 3    | 话数     | 番剧的总话数 |
| 4    | 卷数     | 番剧的总卷数（OVA/剧场版等） |
| 5    | 发售日   | 番剧的首次放送/发售日期 |
| 6    | 评分     | Bangumi网站的综合评分 |
| 7    | 用户评分 | 该用户对番剧的个人评分 |
| 8    | 类型     | 番剧类型（TV/OVA/剧场版等） |
| 9    | 状态     | 番剧在用户收藏中的状态 |
| 10   | 总收藏数 | 该番剧在Bangumi的总收藏人数 |
| 11   | 排名     | 番剧在Bangumi的排名 |


## 使用指南

### 操作步骤

1. **输入Bangumi用户ID**：输入你想要获取清单的Bangumi用户ID
   - 如何获取用户ID？访问Bangumi用户主页，URL中的数字或用户名即为用户ID
   - 例如：`https://bgm.tv/user/jasonshane` 中的 `jasonshane` 就是用户ID

2. **选择收藏类型**：
   ```
   1. 想看
   2. 在看
   3. 已看
   4. 搁置
   5. 抛弃
   ```

3. **选择输出属性**：可多选，用逗号分隔，例如：`1,2,3`

4. **选择输出格式**：
   ```
   1. Excel (.xlsx)
   2. TXT (.txt)
   ```

### 输出文件

程序会在当前目录生成以下格式的文件：
- Excel格式：`bangumi_用户ID_收藏类型_时间戳.xlsx`
- TXT格式：`bangumi_用户ID_收藏类型_时间戳.txt`

## 注意事项

- 本工具仅用于获取公开的Bangumi用户数据
- 请遵守Bangumi API的使用规范，避免过度请求（程序已内置0.5秒请求间隔）
- 部分番剧的某些属性可能为空，这是API返回数据本身的问题
- 如需获取更多番剧信息，可自行扩展代码或提交Issue
